# Permissions

## Public

Can view:

- Home page
- Active public buyer profiles

Cannot view:

- Private contact details
- Verification documents
- Invite records
- Admin routes

## Buyer

Can:

- Manage own buyer profile and criteria
- Upload own profile photo
- View own badge status
- View and respond to own invites
- View own notifications

Cannot:

- Grant badges
- View other users' private documents
- Use seller-only invite actions unless also a seller

## Seller

Can:

- Search active buyer profiles
- View public buyer profiles
- Manage own properties
- Upload own property documents
- Send invites from owned properties
- View own sent invites

Cannot:

- View buyer private financial documents
- Grant badges
- Send unlimited invites
- Invite from properties owned by another user

## Admin

Can:

- Review users, profiles, documents, badges, reports, and invites
- Grant/revoke badges
- Approve/reject documents
- Hide profiles
- Suspend users
- View audit logs

All admin actions that affect trust, visibility, documents, roles, or moderation must write an audit log.
